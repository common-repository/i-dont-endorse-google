== Description ==

Automatically add a rel="nofollow" to all links going to Google's page inside your wordpress posts.

More info:

* More info on [I don't endorse Google](https://www.andreapernici.com/wordpress/nofollow-google/), with info on how to configure it.

== Installation ==

Download, Upgrading, Installation:

Upgrade

1. First deactivate I dont't endorse Google
1. Remove the `nofollow-google` directory

**Install**

1. Unzip the `nofollow-google.zip` file. 
1. Upload the the `nofollow-google` folder (not just the files in it!) to your `wp-contents/plugins` folder. If you're using FTP, use 'binary' mode.

**Activate**

1. In your WordPress administration, go to the Plugins page
1. Activate the "I don't endorse Google" and you're fine.

If you find any bugs or have any ideas, please mail me.
